# zkPetition UI

Next.js frontend for the zkPetition platform.

## Features

- Connect Auro Wallet
- Browse and sign petitions
- Create new petitions
- Zero-knowledge proof generation in Web Worker

## Commands

```bash
npm run dev    # Start development server
npm run build  # Production build
npm run start  # Start production server
npm run lint   # Run linting
```

## Requirements

- [Auro Wallet](https://www.aurowallet.com/) browser extension
- Node.js >= 18.14.0
